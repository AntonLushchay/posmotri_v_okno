# Посмотри в окно

Веб-приложение для просмотра и поиска видео с видами из окон разных городов в разное время суток.

## Описание проекта

Проект представляет собой интерактивную платформу для просмотра видео, снятых из окон в различных городах. Пользователи могут искать видео по названию города и фильтровать их по времени суток.

## Технологический стек

- **HTML5** - семантическая разметка
- **CSS3** - стилизация, flexbox, grid-раскладки
- **JavaScript** (Vanilla JS) - динамическое взаимодействие с API, работа с DOM

## Функциональность

- **Поиск по городу** - возможность найти видео из конкретного города
- **Фильтрация по времени суток** - выбор видео, снятых утром, днем или ночью
- **Воспроизведение видео** - просмотр выбранного видео с постером
- **Выбор из списка** - переключение между видео из списка карточек
- **Пагинация** - загрузка дополнительных видео при нажатии на кнопку "Показать еще"
- **Прелоадер** - индикация загрузки данных
- **Обработка ошибок** - отображение сообщений при отсутствии результатов или ошибках загрузки
